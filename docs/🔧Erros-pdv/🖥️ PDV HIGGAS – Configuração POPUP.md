## 📌 Liberar permissão no terminal

Executar o comando:

chmod +x /usr/local/bin/popup

### 🔎 O que o comando faz:

- `chmod +x` → Concede permissão de execução ao arquivo
    
- `/usr/local/bin/popup` → Caminho do executável do popup

# 🔐 PuTTY

## 📌 Finalidade
Cliente para acesso remoto via SSH e Telnet.

---

## ⬇️ Instalação

1. Baixar o instalador do PuTTY
2. Executar o setup
3. Concluir instalação

---

## ⚙️ Configuração inicial

- Informar IP do servidor
- Porta 22
- Protocolo SSH
- Salvar sessão para reutilização

- Acesso Linux
- Troubleshooting

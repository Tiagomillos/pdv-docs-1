#Procedimento


• Entra na pasta pdvJava

• Crie uma pasta chamada **dados** (ou outro nome de sua preferência):

`mkdir dados`

• Mova os arquivos **.NVL** para a pasta criada:

`mv *.NVL dados/`

• Mova o arquivo **THOOEDCZ.TRA** para a mesma pasta:

`mv THOOEDCZ.TRA dados/`

• Encerre (mate) a aplicação em execução:

`pkill -f nome_da_aplicacao`

_(ajuste o nome da aplicação, se necessário)_

• No PDV, execute a **função 854** para reprocessamento/atualização.
___________________________
Se caso estiver `FECHE A GAVETA` verifique se está fechado a gaveta, e peça para o cliente conectar novamente.

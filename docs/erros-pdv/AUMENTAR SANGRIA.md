## 🔧 Aumentar limite de sangria via Manager

Para ajustar o valor de sangria no sistema, siga o passo a passo:

**Caminho:**  
`Cadastro > Finalizadora`

### Etapas:
1. Filtrar a **loja desejada**  
2. Selecionar a finalizadora **Dinheiro**  
3. Acessar a aba **Validações**  
4. Localizar o campo **Limite de alerta de sangria**  
5. Informar o novo valor conforme solicitado pela loja  
6. Salvar as alterações  

> ✅ Após salvar, o novo limite de sangria já estará ativo para a loja selecionada.

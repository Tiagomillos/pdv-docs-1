# PDV sem subir venda

## Verificação de Licença

• Primeiramente, verificar se o **PDV está licenciado via Manager**  
• Caso não esteja:
  • Ativar a licença do PDV pelo **Manager**
  • Somente após a licença ativa, prosseguir

## Procedimento no PDV

• Acessar o PDV  
• Executar as funções abaixo:

### Função 179 – Recuperar Movimento
• Informar a data no formato **DDMMAA**  
• Exemplo:
  • `260126`  
• Aguardar a finalização do processo

### Função 1255 – Retransmitir
• Executar a função normalmente  
• Aguardar a conclusão

## Resultado Esperado

• Após a retransmissão, o PDV deverá **voltar a subir as vendas corretamente**



Via Manager – Verificação de Vendas do PDV

• Acessar o **Manager**  
• Ir até **Gerenciamento de Lojas**  
• Selecionar a **loja correta**  
• Informar o **dia desejado** para consulta  
• Verificar se as **vendas do PDV foram transmitidas corretamente**

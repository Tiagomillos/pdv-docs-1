# 📂 WinSCP

## 📌 Finalidade
Transferência de arquivos via SFTP.

---

## ⬇️ Instalação

1. Baixar WinSCP
2. Executar instalação
3. Configurar host, usuário e porta

---

## ⚙️ Configuração inicial

- Host/IP do servidor
- Usuário e senha
- Protocolo SFTP
- Porta 22
